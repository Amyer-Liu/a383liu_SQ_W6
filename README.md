# Week 6 - Escape from the village

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

**Controls:**
- Move: WASD
- Shoot: Spacebar (shoots in the direction you last moved)
- Restart: R (after win or game over)

Dodge the dark square obstacles and shoot the sprite mobs. The goal is to score as many points as possible while surviving until the progress bar is completely filled to win the game.

**Opening the Chrome Console**
- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`
